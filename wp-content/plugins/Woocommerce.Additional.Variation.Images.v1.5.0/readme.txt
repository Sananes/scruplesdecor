=== WooCommerce Additional Variation Images ===
Author: Roy Ho
Tags: woocommerce
Requires at least: 3.9
Tested up to: 4.0
Requires WooCommerce at least: 2.1

A WooCommerce plugin/extension that adds ability for shop/store owners to add variation specific images in a group.

See http://woothemes.com/products/woocommerce-additional-variation-images for full documentation.

== Installation ==

1. Upload the entire 'woocommerce-additional-variation-images' folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
